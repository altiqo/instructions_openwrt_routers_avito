Вход в админку для первого подключения

  1. Подключите кабель провайдера в WAN порт роутера.
  2. При необходимости подключите устройство по LAN кабелю (например, в порт LAN1).
  3. Откройте в браузере http://192.168.1.1.
  4. Логин: root, пароль: 12345678.

Как поменять тип подключения

  1. Перейдите в Сеть → Интерфейсы.
  2. Откройте настройки интерфейса WAN.
  3. Выберите протокол PPPoE и введите логин/пароль от провайдера.
  4. Нажмите Сохранить, затем Применить.
  5. Проверьте, что интернет-подключение стало активным.

DHCP: не открывается страница логина провайдера

Позвоните провайдеру и сообщите, что вы заменили роутер. Обычно нужно обновить MAC-адрес на стороне провайдера.

Как поменять название Wi‑Fi и пароль

  1. Откройте Сеть → Беспроводная сеть.
  2. Нажмите Изменить для 2.4G и 5G сетей.
  3. Измените имя сети (SSID) и пароль.
  4. Сохраните и примените настройки.

Как добавить свой сайт в обход

  1. Перейдите в Службы → podkop → Секции.
  2. В разделе «Тип пользовательского списка доменов» выберите Текстовый список.
  3. Добавьте домены в формате ya.ru (без https://).
  4. Примените изменения. Если нужно, перезапустите podkop в разделе «Диагностика».
  5. Если сайт не открывается сразу, проверьте в режиме инкогнито или в другом браузере.

Как перезапустить AdguardVPN

  1. Откройте командную строку (Win + R → cmd).
  2. Подключитесь к роутеру: ssh root@192.168.1.1.
  3. При запросе ключа введите yes, затем пароль роутера.
  4. Запустите команду:
/opt/adguardvpn_cli/adguardvpn-cli connect -l PL

После выполнения команду можно закрыть.

Как заменить изначальную настройку и поставить свой VPN

  1. Откройте Службы → podkop → Секции.
  2. В секции main у вас из коробки стоит AmneziaVPN под блок РКН. Эту секцию не меняем, а добавляем новую. Вам нужно опуститься в самый низ и добавить новую секцию "VPN".
  3. Выберите тип подключения Proxy.
  4. Заполните параметры своего VPN (например, VLESS).
  5. Примените изменения.

Как добавить несколько кофигрураций VLESS VPN?

  1. Откройте Службы → podkop → Секции.
  2. Создаем новую секцию VPN или используем уже существующую.
  3. Выберите тип подключения Proxy.
  4. Тип конфигурации указываем URLTest для автоматического выбора локации с наименьшей задержкой. Для ручного выбора ставим тип Selector.
  5. Вставляем конфигурации VLESS (либо другой протокол из доступных) по очереди в поле Ссылки прокси для URLTest.
  6. Выставляем параметр Интервал проверки URLTest, который отвечает за частоту опроса пинга локаций.
  7. Примените изменения.

Перестал работать YouTube, Instagram и другие сайты в блоке

В случае, если у пропал доступ к сайтам YouTube, Instagram и другим заблокированным РКН, вам нужно попробовать обновить конфиг AmneziaVPN. Как это сделать ниже.

  1. Скачайте любой WARP-конфиг: warp-generator.github.io/warp.
  2. Перейдите в Сеть → Интерфейсы → AmneziaWG и нажмите «Изменить».
  3. Импортируйте содержимое скачанного конфигурационного файла.
  4. Нажмите «Импорт настроек» и примените изменения.

Как прописать DNS для XBOX?

  1. Откройте Службы → podkop → Настройки.
  2. Ставим Тип протокола DNS — UPD.
  3. Переходим на сайт xbox-dns.ru и копируем там Основной (IPv4) и Дополнительный (IPv4).
  4. Прописываем основной DNS в DNS-сервер и дополнительный в Bootstrap DNS-сервер.
  5. Примените изменения.
